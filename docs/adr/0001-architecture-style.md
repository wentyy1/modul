# ADR 0001 — Architecture Style

## Context
Проєкт Peripheral Shop розробляється невеликою командою в рамках навчального курсу.
Поточні вимоги: швидка розробка, простота підтримки, мінімальні інфраструктурні витрати.

## Decision
На цьому етапі обрано **модульний моноліт**.

## Alternatives
Мікросервісна архітектура відкладена як передчасна.

## Consequences
+ Простота розробки і деплою  
+ Висока швидкість змін  
− Менш гнучке масштабування

## Review Conditions
Перегляд рішення при зростанні команди або навантаження.
