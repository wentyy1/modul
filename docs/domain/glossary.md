# Ubiquitous Language — Peripheral Shop

- **Product** — товар периферії (клавіатура або мишка)
- **Catalog** — сукупність усіх доступних товарів
- **User** — зареєстрований користувач системи
- **Admin** — користувач з правами керування товарами
- **Order** — замовлення одного або кількох товарів
- **OrderItem** — позиція в замовленні
- **Price** — вартість одиниці товару
- **Status** — поточний стан замовлення
