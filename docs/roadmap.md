# Peripheral Shop — Roadmap

## Найближчі 3–5 тижнів

### Тиждень 3
- Розробити API endpoint `/products` для отримання списку товарів
- Підключити просту базу даних

### Тиждень 4
- Реалізувати базову авторизацію користувачів
- Створити систему ролей (User, Admin)

### Тиждень 5
- Реалізувати CRUD для товарів
- Створити адміністративну панель керування товарами
